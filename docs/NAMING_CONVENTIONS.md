
NAMING CONVENTIONS
kebab-case for folders: components/user-profile/

PascalCase for React components: UserProfile.tsx

camelCase for variables/functions: getUserData()

PascalCase for TypeScript interfaces: interface UserProps

UPPER_SNAKE_CASE for constants: MAX_RETRY_COUNT

File Naming
page.tsx, layout.tsx, ComponentName.tsx, route.ts

Database
Models: PascalCase singular (User, Product)
Fields: camelCase (firstName, createdAt)
Enums: PascalCase (UserRole, OrderStatus)
